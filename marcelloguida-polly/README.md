# Polly — Marketing Website

Static marketing site for the Polly iOS app, published under the portfolio at
`https://marcelloguida00.github.io/marcelloguida-polly/`.

| Requirement           | File                         | URL |
|------------------------|------------------------------|-----|
| Marketing URL          | `indexPolly.html`            | `/marcelloguida-polly/indexPolly.html` |
| Support URL            | `supportPolly.html`          | `/marcelloguida-polly/supportPolly.html` |
| Privacy Policy URL     | `privacyPolly.html`          | `/marcelloguida-polly/privacyPolly.html` |
| Data Management URL    | `data-managementPolly.html`  | `/marcelloguida-polly/data-managementPolly.html` |

## Project structure

```
marcelloguida-polly/
├── indexPolly.html
├── supportPolly.html
├── privacyPolly.html
├── data-managementPolly.html
├── 404Polly.html
├── css/style.css
├── js/main.js
├── assets/
├── robots.txt
├── sitemap.xml
└── site.webmanifest
```

Source lives in `marcelloguida-polly/`. The portfolio build copies it to
`public/marcelloguida-polly/` via `npm run sync:polly`.

## Local preview

```bash
npm run sync:polly
npx serve public/marcelloguida-polly
```
